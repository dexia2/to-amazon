to-amazon
===============

## 機能

+ 文字を選択してコンテキストメニューを開くことで、キーワードのAmazonの検索ページを別タブで開きます
	+ 似た拡張は他にもありますが、日本語ページに飛びたかったので
+ Amazonの商品ページにコピーボタンを設置し、情報をクリップボードに保存します
